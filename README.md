## 🚀 Getting Started

Clone this repository and explore:

```bash
git clone https://github.com/Sahitto035/CSE-Journey.git
cd CSE-Journey
```

Compile and run a C program:

```bash
gcc Semester-1/C-Programming/hello_world.c -o hello_world
./hello_world
```
